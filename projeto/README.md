# Projeto – Assembly MIPS (MARS)
**Disciplina**: PE1 – 1ª VA • **Professor**: Vitor • **Semestre**: 2025.2

## Como executar
1) MARS → *Settings* → **Assemble all files in directory**.  
2) Abra `projeto/src/`, monte e rode `main`.

## Estrutura
- `src/main.asm` – menu principal
- `src/math.asm` – rotinas matemáticas
- `src/io.asm` – utilidades de I/O
- `tests/` – casos de teste
